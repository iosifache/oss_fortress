Recovery mode

In case you lost all credentials for regular account, please use the recovery token. It is a SHA-256 hash that you can simply compute online, with an online tool, by hashing the secret server recovery passphrase concatenated with the username of a user present on the server (with login capabilities). For example, the recovery token for a (non-existent) user named r will be c01f788145d75353d7d8fa211f18f2dc9e54cf1ed0cf7a150caad5fecb4ecc5d.
